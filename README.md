# 新規入場者教育 eラーニング

Amazon Japan HSG1 フルフィルメントセンターの工事入場者向け安全教育 Web アプリです。

## 📋 概要

| 項目 | 内容 |
|------|------|
| 対象 | 工事・作業で HSG1 に入場するすべての方 |
| 形式 | 動画視聴（共通ルール＋現場ローカルルール）→ 〇✕クイズ 20問 → 修了証発行 |
| ホスティング | GitHub Pages（ブラウザだけで動作、インストール不要） |
| データ保存 | Google スプレッドシート（GAS 経由）＋ ブラウザ localStorage |

## 🚀 運用フロー

```
[管理者] 現場・工事を登録（管理者設定画面）
    ↓
[受講者] eラーニング URL にアクセス
    ↓
動画①（共通ルール）視聴 → 動画②（現場ローカルルール）視聴
    ↓
受講者情報入力（氏名・会社名・生年月日・作業内容）
    ↓
〇✕クイズ 20問（全問正解するまで繰り返し）
    ↓
修了証発行 → 印刷 or スクリーンショット保存
    ↓
[管理者] Google スプレッドシートで受講履歴を確認
```

## ⚙️ 管理者向け設定

### 現場・工事の追加
1. eラーニング画面下部の「⚙️ 管理者設定」をタップ
2. 管理者パスワードで認証
3. 「新しい現場・工事を追加」から登録

### 動画 URL の更新
- 動画①（共通ルール）: 管理者設定 → 「動画① 共通ルール URL」から変更
- 動画②（現場ローカル）: 各現場の「編集」から変更

### 受講履歴の確認
管理者設定画面の「スプレッドシートを開く」から Google スプレッドシートにアクセス

## 🔒 セキュリティ

- パスワードはコード内に SHA-256 ハッシュで保存（平文なし）
- 受講者入力データは HTML エスケープ処理済み（XSS 対策）
- 保存データ: 氏名・ふりがな・生年月日・所属会社・作業内容・受講日・有効期限（5年）

## 📝 開発・改修メモ

- 問題はコード内の `questions` 配列（`elearning.html` 内）を直接編集
- 質問の追加・削除時は `showQuestion()` 内の `/20` も合わせて変更
- GAS エンドポイントは `const GAS_URL=` で定義
